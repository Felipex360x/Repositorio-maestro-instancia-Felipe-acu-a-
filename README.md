# Repositorio-maestro-instancia-Felipe-acu-a-

en este repositorio se encuentran todo lo que este relacionado con la estancia de felipe acuña
## Microservicios y repositorios

| Servicio       | Encargado | Repositorio                                    |
| :------------- | :-------- | :--------------------------------------------- |
| MS-Usuario     | Felipe    | https://github.com/Felipex360x/ms-usuario     |
| MS-Comprador   | Felipe      | https://github.com/equipoN/pago-service        |
| MS-Vendedor     | Felipe     | https://github.com/equipoN/producto-service    |
| MS-Repartidor    | Felipe     | https://github.com/equipoN/orden-service       |
| MS-Vehiculo      | Felipe | https://github.com/equipoN/reporte-service     |
| MS-Envio      | Felipe | https://github.com/equipoN/reporte-service     |

## Despliegue
Cada integrante levanta su servicio en su propia EC2 con
`docker compose up -d`.
